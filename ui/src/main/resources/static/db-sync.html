Database Mode

Импорт модулей из файлов

Синхронизация файловых модулей из appsRoot в PostgreSQL registry. Создание или обновление модулей на основе application.yml.

Режим: FILES К DB-модулям
DB-режим недоступен
Для импорта нужен активный DB-режим.
Импорт модулей

Импорт сканирует appsRoot директорию, находит модули с application.yml, и создаёт/обновляет их в PostgreSQL registry. Модули с совпадающим content_hash пропускаются.

Результат синхронизации
Разработано командой MLP
